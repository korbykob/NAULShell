# NAUL Shell
A shell program for the NAUL operating system.

## Usage
| Command      | Description              |
| ------------ | ------------------------ |
| `help`       | Shows these commands     |
| `exit`       | Exit the shell           |
| `ping`       | Reply with "Pong!"       |
| `clear`      | Clear the terminal       |
| `kys`        | Panic the system         |
| `(folder)/*` | List files in the folder |
| `(folder)/`  | Enter the folder         |
| `..`         | Go back a folder         |
| `(file)`     | Execute or read the file |
